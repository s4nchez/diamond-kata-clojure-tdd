# Diamond Kata 

My implementation of the diamond kata using TDD and Clojure.

A detailed explanation about this solution can be found  [here](http://www.isanchez.net/diamond-kata-using-clojure-and-tdd.html).

## Usage

Running:

```clojure
(print-diamond "F")
```

Produces
```
-----A-----
----B-B----
---C---C---
--D-----D--
-E-------E-
F---------F
-E-------E-
--D-----D--
---C---C---
----B-B----
-----A-----
```
